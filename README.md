# spinning-top

# unity結合arduino

> * 使用arduino+Mpu6050加速度計與陀螺儀感測器當成陀螺的訊號輸入。
> * 兩邊甩玩陀螺後pk陀螺，數值與陀螺轉速損耗取決於Mpu6050感測測值。
> * 每次陀螺碰撞減少轉速並隨機減少穩定定。
